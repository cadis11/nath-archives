⚙️ One-Time GitHub Setup (PowerShell 7)

  1. Install Git and GitHub CLI (winget install GitHub.cli)
  2. Create a new public repository named nath-archives on GitHub
  3. Clone this project and push it to your repo
  4. Enable GitHub Pages: Settings → Pages → Source: main branch / root
  5. Your archive will be live at: https://YOUR-USERNAME.github.io/nath-archives/

🚀 Initial Deploy — PowerShell 7 Script

Run this in PowerShell 7 after downloading the archive files:

# Run in PowerShell 7 # ----------------------------------------------- # 1. Set your GitHub username $USERNAME = "your-github-username" $REPO = "nath-archives" # 2. Init and push git init git add . git commit -m "Initial commit: Nath Digital Archives" git branch -M main git remote add origin "https://github.com/$USERNAME/$REPO.git" git push -u origin main # 3. Enable GitHub Pages via GitHub CLI gh auth login gh api repos/$USERNAME/$REPO/pages --method POST ` --field source='{"branch":"main","path":"/"}' ` --silent Write-Host "✓ Archive deployed! Visit: https://$USERNAME.github.io/$REPO/" -ForegroundColor Green

🔑 Generate GitHub Token for Uploads

  1. Go to github.com/settings/tokens/new
  2. Note: nath-archives-admin
  3. Expiration: 90 days (renew as needed)
  4. Scope: check repo (full control)
  5. Click Generate token — copy and save it
  6. Paste this token in the Upload tab before uploading files

🔐 Change Admin Password

Edit admin/index.html and find the ADMIN_CREDENTIALS section near the top of the script:

// Find this in admin/index.html and change: const ADMIN_USER = 'keeper'; // ← change username const ADMIN_PASS = 'gorakh108'; // ← change password

⚠️ Note: Since this is a static site, credentials are in the HTML. For stronger security, keep the admin page private or use a private GitHub repo with Pages.

📤 Bulk Upload — PowerShell 7

To batch-upload many PDFs at once:

# Bulk upload all PDFs from a folder $TOKEN = "ghp_your_token_here" $REPO = "your-username/nath-archives" $FOLDER = "C:\NathBooks" Get-ChildItem "$FOLDER\*.pdf" | ForEach-Object { $file = $_.FullName $name = $_.Name $b64 = [Convert]::ToBase64String([IO.File]::ReadAllBytes($file)) $body = @{ message = "Add book: $name" content = $b64 } | ConvertTo-Json Invoke-RestMethod ` -Uri "https://api.github.com/repos/$REPO/contents/books/$name" ` -Method PUT ` -Headers @{ Authorization = "token $TOKEN" "Content-Type" = "application/json" } ` -Body $body Write-Host "✓ Uploaded: $name" -ForegroundColor Green }
